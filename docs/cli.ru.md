# Справка по CLI

SILO включает в себя несколько утилит командной строки для упрощения разработки.

## `silo init`
Создание нового навыка из минимального шаблона.

**Использование:**
```bash
silo init <filename> [--secrets SEC1,SEC2]
```

**Параметры:**
*   `<filename>`: Имя файла, например `github_skill.py`.
*   `--secrets`: Список необходимых ключей через запятую, например `GITHUB_TOKEN,OPENAI_API_KEY`.

Команда создает файл, соответствующий `PEP 723`, что позволяет `uv` запускать его со всеми зависимостями автоматически.

## `silo test`

Запуск навыка в симуляции "безголового" агента.

**Использование:**
```bash
silo test <script> [command] [--args...]
```

**Зачем это нужно?** ИИ-агент не видит цветов, прогресс-баров и не может нажать "ОК" в терминале. Ему нужен чистый текст или JSON. `silo test` заставляет SILO работать в режиме Headless. Команда проверяет генерацию манифеста (`SKILL.md`), работу авторизации без зависаний и чистоту вывода.

## `silo doctor`

Диагностика окружения.

**Использование:**
```bash
silo doctor
```

Проверяет:
*   Версию Python (нужно 3.9+).
*   Наличие `uv` (рекомендуется).
*   Интеграцию с системным Keychain.
*   Критические зависимости (например, `pydantic`).
