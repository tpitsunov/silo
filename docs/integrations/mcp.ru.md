# Деплой через MCP

Model Context Protocol (MCP) от Anthropic — это открытый стандарт, позволяющий разработчикам создавать защищенные серверы, предоставляющие инструменты и интеграции для ИИ-интерфейсов (например, приложения Claude Desktop).

SILO бесшовно поддерживает экспорт ваших скиллов в качестве MCP-сервера.

## Как это работает

Вам не нужно переписывать логику вашего приложения для поддержки MCP. 

Просто запустите ваш скилл через встроенную команду MCP:

```bash
silo mcp run
```

## Настройка Claude Desktop

Чтобы добавить ваш скилл в Claude Desktop, укажите путь к нему в конфигурационном файле (например, `~/Library/Application Support/Claude/claude_desktop_config.json` на macOS):

```json
{
  "mcpServers": {
    "my_silo_skill": {
      "command": "silo",
      "args": [
        "mcp",
        "run",
        "--skill",
        "my-skill"
      ]
    }
  }
}
```

Благодаря тому, что SILO использует PEP 723 для декларации инлайновых зависимостей, Claude Desktop автоматически установит все необходимое в изолированную среду при запуске, обеспечивая идеальную работу ваших инструментов.

Все правила SILO (пропущенные токены или необходимость подтверждения действий) будут работать "из коробки" через локальные уведомления и браузер, обеспечивая безопасность вашего окружения.
