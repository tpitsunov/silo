# Справочник CLI

SILO предоставляет набор команд для управления вашими скиллами.

## `silo init`
Создать структуру для нового скилла.

**Использование:**
```bash
silo init <namespace>
```

## `silo install`
Установить локальную папку со скиллом в хаб SILO.

**Использование:**
```bash
silo install <путь_к_папке_скилла>
```

## `silo remove`
Удалить скилл и надежно стереть его секреты из системного Keychain.

**Использование:**
```bash
silo remove <namespace>
```

## `silo ps`
Показать список всех установленных скиллов в хабе.

**Использование:**
```bash
silo ps
```

## `silo run`
Запустить конкретный инструмент (tool) внутри скилла.

**Использование:**
```bash
silo run <namespace> <tool_name> [аргументы...]
```

## `silo inspect`
Показать инструкции и доступные инструменты для конкретного скилла.

**Использование:**
```bash
silo inspect <namespace>
```

## `silo mcp run`
Запустить роутер MCP (Model Context Protocol) для подключения ваших скиллов к ИИ-агентам (например, Claude Desktop).

**Использование:**
```bash
silo mcp run
```

## `silo prune`
Удалить все установленные скиллы из хаба.

**Использование:**
```bash
silo prune
```
