# SILO Framework

Добро пожаловать в документацию SILO — надежного слоя исполнения для инструментов ИИ.

## Что такое SILO?

SILO — это надежный движок исполнения и менеджер пакетов для инструментов (скиллов) ИИ-агентов. В отличие от традиционных фреймворков, SILO делает упор на:

* **🔒 Изоляция**: Каждый скилл запускается в собственном виртуальном окружении `uv`.
* **🛡️ Безопасность**: Секреты шифруются с помощью AES-256-GCM и передаются через STDIN.
* **🧩 Динамическая маршрутизация**: Агенты ищут инструменты семантически с помощью BM25 и точного совпадения.
* **✨ Премиальный DX**: Подтверждения через браузер и интерактивные потоки авторизации.

## Начало работы

Установите основной фреймворк:

```bash
uv tool install silo-framework
```

Инициализируйте ваш первый скилл:

```bash
silo init my-skill
```

## Почему SILO?

SILO помогает разработчикам воплощать идеи в жизнь, укрощая сложность создания инструментов для ИИ. Он предоставляет доверенную точку входа для каждой интеграции агента, гарантируя, что инструменты будут легко обнаруживаемыми, безопасными и воспроизводимыми в любой среде.
